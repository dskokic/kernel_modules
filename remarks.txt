To be able to build linux kernel module you need to install linux kernel header package:

Ubuntu: 	> sudo apt install linux-headers-$(uname -r)
Raspberry Pi:	> sudo apt install raspberrypi-kernel-headers

This will install kernel header source file in directory /usr/src/linux-headers-$(uname -r) folder,
and will create soft link to it, named build in /lib/modules/$(uname -r) folder.

e.g.:
if kernel version is 5.10.11-v7l+ (uname -r)

then
/lib/modules/5.10.11-v7l+/build 
is link to 
-> /usr/src/linux-headers-5.10.11-v7l+
